# 📦 YZAT Almoxarifado

Sistema de gerenciamento de almoxarifado desenvolvido em Node.js.

## Funcionalidades

- 🔐 Login do sistema
- 📦 Cadastro de produtos
- ✏️ Edição de produtos
- ➕ Entrada de estoque
- ➖ Saída de estoque
- 🗑️ Exclusão de produtos
- 🔍 Pesquisa em tempo real
- 📊 Dashboard com gráficos
- 📱 Aplicativo PWA (instalável)
- ☁️ Versionamento com GitHub

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- Chart.js
